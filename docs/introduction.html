Creating an NFFS SYmposium Article Using LaTeX¶

This page documents the process i used to generate an NFFS article using LaTeX and pandoc. All of this material is available on the companion website at https://rblack42.github.io/nffs-workflow.

Step 1 - Create a simple LaTeX doocument¶

I started off with a simple document:

\documentclass[12pt,twocolumn]{article}
\usepackage{../assets/nffs}

\author{Roie R. Black}
\date{\today}
\title{Predicting indoor Model Flight Time}
\titlepic{\includegraphics[width=0.66\textwidth]{math-magik-lpp.png}}


\begin{document}
\maketitle

\input{content/introduction}
\input{content/biography}
\bibliographystyle{abbrvnat}
\bibliography{references.bib}
\end{document}
